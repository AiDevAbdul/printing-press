### Setup & Installation
```bash
# First time setup
setup.bat  # or ./setup.sh

# Manual setup
cd backend && npm install && npm run build && npm run migration:run

```

### Daily Development
```bash
# Start backend
cd backend && npm run start:dev

# Start frontend
cd frontend && npm run dev
```

### Database Management
```bash

npm run migration:revert
```
