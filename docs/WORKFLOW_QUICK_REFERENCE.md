# Production Workflow - Quick Reference Card

## 🎯 Quick Actions

### Starting a Stage
1. Enter your **Operator ID**
2. Enter **Machine Name** (e.g., HB2, UV#1)
3. Click **▶ START**

### Pausing Work
1. Click **⏸ PAUSE**
2. Enter reason (optional)
3. Click **Pause** to confirm

### Resuming Work
1. Click **▶ RESUME**
2. Work continues immediately

### Completing a Stage
1. Click **✓ COMPLETE**
2. Enter waste quantity (optional)
3. Add notes (optional)
4. Click **Complete** to confirm

## ⏱️ Time Tracking

- **Active Time** = Work time (excludes pauses)
- **Pause Time** = Break time (tracked separately)
- **Total Time** = Active time only (shown when complete)

**Example:**
```
Start: 2:00 PM
Pause: 2:30 PM (worked 30 min)
Resume: 3:00 PM (paused 30 min)
Complete: 3:45 PM (worked 45 min more)
Total Active Time: 75 minutes ✅
```

## ⚠️ Important Rules

✅ **DO:**
- Complete stages in order
- Record waste quantity
- Add notes for issues
- Enter pause reasons
- Wait 1 minute before completing

❌ **DON'T:**
- Skip stages
- Complete without checking quality
- Forget to record waste
- Leave notes empty for issues

## 🔧 Common Machines

- **HB1, HB2** - Heidelberg printing machines
- **UV#1, UV#2** - UV coating machines
- **Dye 1, Dye 2** - Die cutting machines
- **Lam 1** - Lamination machine

---

## 📞 Need Help?

1. **Can't start stage?** → Check previous stage is complete
2. **Button disabled?** → Wait for previous stage or refresh page
3. **Duration not updating?** → Page refreshes every 5 seconds
4. **Made a mistake?** → Contact supervisor immediately

---

## 💡 Pro Tips

- Have operator ID ready before starting
- Record waste immediately after completion
- Use pause for breaks, not for switching jobs
- Add notes for any quality issues
- Check machine name is correct before starting
