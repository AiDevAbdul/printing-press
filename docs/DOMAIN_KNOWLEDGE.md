## GSM (Grams per Square Meter)

Weight measurement for paper/card. Common values:
- 80 GSM - Thin paper (standard copy paper)
- 100 GSM - Medium paper
- 150 GSM - Thick paper
- 200 GSM - Card stock
- 250 GSM - Heavy card stock
- 300 GSM - Premium card stock

## Printing Processes

### Color Printing
- **CMYK** - Cyan, Magenta, Yellow, Black (standard 4-color process)
- **Pantone** - Spot colors (PMS colors)
- **Four Color Process** - Full CMYK printing

### Special Finishes
- **CTP** - Computer-to-Plate (direct plate imaging)
- **Spot UV** - Selective UV coating for shine/protection
- **Plain UV** - Full UV coating
- **Drip-Off UV** - UV coating with dripping effect
- **Metalize** - Metallic coating/foil
- **Emboss** - Raised/textured effect
- **Lamination** - Protective plastic layer

## Measurements

### Length & Width
- Measured in millimeters (mm)
- Standard sizes: A4 (210×297mm), A3 (297×420mm), etc.

### Weight Calculation

**For Cards** (per packet of 100 sheets):
```
Weight (kg) = (Length × Width × GSM) / 15500
```

**For Paper** (per ream of 500 sheets):
```
Weight (kg) = (Length × Width × GSM) / 3100
```

## Pricing Components

### Material Cost
- **Price per KG** - Cost of raw material
- **Conversion %** - Waste/conversion factor (typically 5-15%)

### Fixed Charges
- **CTP Charge** - Computer-to-Plate setup fee
- **Spot UV Charge** - Special UV coating fee
- **Plain UV Charge** - Full UV coating fee
- **Drip-Off UV Charge** - Drip-off UV effect fee
- **Metalize Charge** - Metallic coating fee
- **Emboss Charge** - Embossing setup fee
- **Lamination Charge** - Lamination fee
- **Others** - Miscellaneous charges

### UPS (Unit Per Sheet)
- Number of units printed per sheet
- Affects total quantity and cost

## Production Workflow Stages

| Stage | Description | Typical Duration |
|-------|-------------|------------------|
| Pre-Press | Design preparation, file setup | 1-2 days |
| Plate Making | Create printing plates | 1 day |
| Machine Setup | Configure printing machine | 2-4 hours |
| Printing | Main printing process | 1-3 days |
| Finishing | Post-print processing (optional) | 1-2 days |
| Quality Check | QA inspection (optional) | 4-8 hours |
| Dispatch | Ready for delivery | 1 day |
